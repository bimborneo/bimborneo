Revit API & BIM Automation

True efficiency in Digital Construction goes beyond merely using software—it's about engineering your own solutions when standard tools fall short.

Driven by a problem-solving mindset, I built these exclusive "BIM Borneo Tools" to tackle real-world modeling bottlenecks, saving countless hours of manual work. What you see here is merely a fraction of my ongoing automation projects. I am constantly developing new logics and plugins to push the boundaries of what BIM can achieve.

Integrity BIM 5D Estimator

C# Revit API

[ PROBLEM ] Calculating project costs manually outside of Revit often leads to discrepancies between the BIM model and the Bill of Quantities (BoQ), requiring time-consuming revisions.

[ SOLUTION ] This tool automatically injects AHSP (Unit Price Analysis) parameters directly into Revit elements and synchronizes prices in real-time, fully automating the 5D Cost Estimation process.

Proprietary Plugin

Bridge & Highway Alignment

C# Revit API

[ PROBLEM ] Manually placing adaptive components (like bridge segments) along a complex Civil 3D alignment inside Revit is highly inaccurate and tedious.

[ SOLUTION ] Reads alignment data coordinates (C3D) and automatically places adaptive components with absolute precision along the predefined curves and elevations.

Proprietary Plugin

Support Tools (Auto Join & Clash)

C# Revit API

[ PROBLEM ] Unjoined intersecting geometry causes inaccurate volume calculations (QTO), and finding overlapping elements manually takes hours.

[ SOLUTION ] Automates the geometry joining process between structural elements and provides an instant 'Clash Detective' to highlight overlapping warnings.

Proprietary Plugin

Rebar Visibility Automator

C# Revit API

[ PROBLEM ] Setting structural rebar to be visible as "Solid" in 3D views requires selecting individual rebar sets and modifying properties over and over.

[ SOLUTION ] A single-click execution tool that instantly overrides the visibility graphics of all reinforcement elements in the active 3D view to be solid and unobscured.
